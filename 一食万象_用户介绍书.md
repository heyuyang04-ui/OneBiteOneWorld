# 一食万象 · One Bite, One World

## 产品介绍书

> 一箪食中遇自己，一味之间找彼此，一城烟火见众生。

---

## 一、为什么是"一食万象"

人生大约有三万顿饭。每一顿，你都在做选择——吃什么、和谁吃、在哪里吃。但你可曾想过：这些选择加起来，到底说了你是一个怎样的人？

我们习惯了用照片记录旅行、用日记记录心情，却从未认真记录过自己吃了什么。那些散落在相册、外卖订单、支付记录里的饮食碎片，其实藏着一个你自己都未曾认识的你。

**一食万象，就是你的 Agent，帮你从每一顿饭中，重新认识自己、遇见他人、读懂城市。**

---

## 二、你的每一顿饭里，藏着什么秘密

### 你比你想象中更"好懂"

你有没有过这样的时刻——打开外卖 App 翻了十分钟，最后还是点了那家店？

那不是选择困难症。那是你的味觉偏好已经形成了一套稳定的"密码"。一食万象的 Agent 通过你拍照上传的食物，持续学习你的口味偏好，构建一个属于你的 **"味觉数字孪生"**。

它知道你喜欢的是"微辣的川菜"而不是"麻辣的湘菜"，知道你在周一喜欢清淡、周五偏爱放纵，知道你在下雨天会比晴天多吃 30% 的碳水。这些不是它猜的，而是它从你日复一日的饮食中"看见"的。

### 你吃的，就是你的生活

Agent 不只是识别菜名。它会给你每周发出一份**味觉周报**，告诉你：

- 这周你的饮食比你想象中健康得多——蔬菜摄入比上周提升了 22%
- 你最近三周的口味正在从"咸鲜"向"酸甜"迁移，这可能和你开始健身有关
- 注意到你这周三次在深夜 11 点后进食，比你的平均水平高出很多——最近压力大吗？

这些不是冷冰冰的数据，而是 Agent 帮你从食物中发现自己的方式。

### 从看见到预见

更妙的是，Agent 可以**预测你下一步会喜欢什么**。基于你对 47 次火锅、23 次日料、18 次越南菜的偏好演变，它会告诉你：

> "根据你的味觉轨迹，下个月你可能会喜欢上贵州酸汤鱼。一家评分很高的店刚好在你公司附近开门了。"

这不再是你去找好吃的，而是好吃的**主动找到了你**。

## 三、世界上有没有另一个你，和你的舌头共享同一套密码

### 以味寻人

你有没有好奇过：有没有人和我有完全一样的口味？我们点的外卖高度重合，对香菜的恨同样强烈，对某个小店的某道菜有完全相同的心动？

在传统社交产品里，你们永远不会相遇。因为算法只看你们的兴趣标签——"美食爱好者"，世界上有几亿个。

但一食万象的 Agent，会在脱敏后分析你真实的味觉数据。它会找到那个**不吃香菜、爱点中辣、日料取向和你 89% 相似、曾和你去过同一家小面馆**的人。然后告诉你：你们的口味交集在这里，差异在那里；如果你们一起吃顿饭，Agent 推荐这几道菜作为起点。

这就是**味觉灵魂伴侣**。它不基于照片、自我介绍、兴趣爱好——只基于你们的舌头。

### 两个 Agent 的对话

更神奇的是，当你的 Agent 和另一个人的 Agent 相遇时，它们会互相"对话"。不是人式的寒暄，而是数据式的交流：

> 你的 Agent："我的用户最近偏爱酸辣口，日料频次在降低。"  
> 对方的 Agent："我的用户也在经历类似的口味迁移，他们的交集菜系是东南亚料理。"  
> 两个 Agent 共同："推荐你俩去试试 XX 餐厅的新菜'青柠酸辣鲈鱼'——你俩都会喜欢。"

两个素未谋面的人，因为 Agent 之间的对话，坐在了同一张餐桌前。这是传统社交产品做不到的事。

## 四、一座城在吃什么，就是一座城在想什么

### 城市也有味觉

一个人有自己的口味演变，一座城也有。

一食万象聚合了千万用户的味觉数据后，可以生成城市的**味觉热力图**。你可以看到：

- 辣味正在从重庆、成都向东移动——杭州、上海近两年的辣味饮食增长了 47%
- 一线城市的工作日午餐时间在缩短——从 2019 年的平均 38 分钟降至今年的 22 分钟
- 每年立秋那天，"贴秋膘"相关饮食会在下午 4 点开始集中爆发

这不是问卷调查报告，而是从真实的、每一天的餐桌上汇聚出来的**城市脉搏**。

### 当你来到一座陌生城市

你出差到成都，打开一食万象的城市视角——

Agent 不会给你"成都必吃的 10 家店"这种大众榜单。它会说：

> "欢迎来到成都。我分析了你的 218 次火锅记录，和这座城市的 12 万家餐厅数据。你应该从宽窄巷子旁边的'老味道'开始——它和你最爱的重庆那家有 73% 的味觉相似度，而且有 7 成成都本地人给它的评分超过你的偏好阈值。"

你不是在吃成都，你是在吃**属于你的成都**。

### 从吃饭到理解时代

当一个城市 50 万人的口味数据汇聚在一起，Agent 可以看到更多：

- 某街区的外卖订单中，蔬菜比例连续下降 3 个月——这可能意味着该区域需要更多的生鲜供应
- 某季节某类食物的搜索量激增——这可能和一部热门剧集中出现了这道菜有关
- 全城的糖分摄入在过去 5 年持续上升——这是公共健康领域需要关注的信号

Agent 从食物中，读出了一个时代的情绪与变迁。

## 五、一食万象的产品哲学

### 不是工具，是伙伴

传统产品的核心是"功能"，一食万象的核心是**关系**。

你用的越久，Agent 越懂你。它不只是在你打开 App 时才存在——它会主动找到你，在合适的时候告诉你合适的事。就像朋友会在你迷茫时给建议，在你疲惫时给你安慰。

### 不是记录，是理解

一食万象不只是在"记"你吃了什么。它是在**理解**你为什么吃这些、这些选择意味着什么、你在发生怎样的变化。

"记录"的终点是一份清单，"理解"的终点是一次觉醒。

### 不是一个人的事，是一群人的事

你的每一条数据，不只是让你更懂自己。它也成为了城市味觉智能的一部分。而当城市数据反过来为你提供洞察时，你就成为了一个更大图景的一部分。

**这就是"一"到"万"的飞轮：一份午餐照见了一个人，人与人连接成社群，社群汇聚成城市，城市反映着时代。**

## 六、谁在用一食万象

- **阿杰，26 岁，杭州程序员**：靠外卖"续命"三年了。用一食万象后，Agent 告诉他："你过去三个月蛋白质摄入下降了 15%，碳水上升了 22%。要看看你公司附近评分超过 4.5 的健康餐吗？"——他开始重视自己吃了什么。

- **小雅，29 岁，上海自由插画师**：把吃当作创作灵感。Agent 帮她发现了口味轨迹和创作主题之间的隐秘关联——"你在画温暖治愈系作品最多的月份，甜食和日料的占比明显偏高。"

- **老周，33 岁，成都餐饮创业者**：通过城市味觉地图发现，自己店周边的区域"酸辣"偏好正在超过"麻辣"，于是调整了菜单结构，三个月后营业额增长了 40%。

## 七、如何使用一食万象

### Step 1：随手一拍
吃饭前拍张照，Agent 自动识别菜品、营养成分，你的味觉档案就更新了。没有表格，没有标签，没有分类负担。

### Step 2：等待洞察
每周日晚上，Agent 会主动推送你的味觉周报。不用天天盯着，让它自己累积和思考。

### Step 3：发现惊喜
Agent 不定期告诉你："有个人和你的口味 91% 相似"；"你的味觉轨迹正在发生一个有趣的转变"；"这座城市最近发生了一件和食物有关的趣事"……

### 越用越准，越用越懂你。

## 八、一食万象的承诺

- **你的数据属于你**：你可以随时查看 Agent 知道了你什么、删掉任何不想保留的记录。
- **社交基于脱敏**：味觉匹配不暴露你的具体饮食记录，只基于抽象的"味觉向量"。
- **Agent 服务于你**：不是推荐算法在操控你，而是一个真的希望帮你看清自己的伙伴。

我们相信，认识自己不一定需要深山修行，也可以从今天的午餐开始。

我们相信，数据不应该是冷冰冰的报表，而应该是一面镜子——照见你、照见彼此、照见我们共同生活的这座城市。

**一食万象。从你的下一顿饭开始，看见更大的世界。**
